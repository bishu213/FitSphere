FitSphere — Smart Workout Trainer

FitSphere is a real-time AI workout tracker built with TensorFlow.js MoveNet.
It detects exercises through your webcam, counts reps, calculates calories burned, and even talks to you with voice feedback.
All sessions are stored locally and visualized in a clean analytics dashboard.

- Features

- Real-time Pose Detection using TensorFlow.js MoveNet

- Automatic Rep Counting for multiple exercises

- Voice Feedback — start, stop, rep count, and summaries

- Form Accuracy Tracking

- Calories Burned Calculation (per rep)

- Dashboard with session history and performance charts

- Local Session Storage (no backend needed)

- Modular JS Architecture
